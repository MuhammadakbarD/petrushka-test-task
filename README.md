##  Исправленная версия ТЗ

### Функционал корзины

1. Пользователь может добавить **1–10 единиц одного товара**
2. Можно изменять количество **от 1 до 10**
3. Удаление — кнопка **"Удалить"**
4. Максимум **5 разных товаров**
5. Максимум **20 товаров всего**

---

### Ограничения

| Тип ограничения | Значение |
| --------------- | -------- |
| Один товар      | до 10    |
| Разные товары   | до 5     |
| Всего товаров   | до 20    |

---

### Ошибки

* "Нельзя добавить более 10 единиц"
* "Максимум 5 товаров"
* "Максимум 20 товаров в корзине"

---

### Цена

✔ Цена фиксируется при добавлении в корзину

---

### Отображение

* название
* количество
* цена
* сумма
* итог

---

### Реклама

* только **будние дни**
* **08:00–11:00**
* **18:00–21:00**
* максимум **3 товара**

---

#  Задание 3 — Архитектура PUSH

## Архитектура

```
Mobile App
    ↓
API Gateway
    ↓
Notification Service
    ↓
Message Broker (Kafka / RabbitMQ)
    ↓
Push Service
    ↓
FCM / APNS
    ↓
User Device
```

---

## Как работает

1. Событие:

   * брошенная корзина
   * отмена заказа
   * реклама

2. Событие → очередь

3. Notification Service обрабатывает

4. Отправка через:

   * Firebase (Android)
   * APNS (iOS)

5. Пользователь получает PUSH
